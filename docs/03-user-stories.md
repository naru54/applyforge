# User Stories

## Authentication

* As a user, I want to register an account so that I can securely access ApplyForge.
* As a user, I want to log in so that I can manage my job applications.

## Resume Management

* As a user, I want to upload my resume so that the system can analyze my profile.
* As a user, I want to replace my resume whenever I update it.

## Job Matching

* As a user, I want to compare my resume with a job description so that I know whether the job suits my profile.
* As a user, I want to see my matching skills and missing skills.

## Job Tracking

* As a user, I want to save interesting jobs.
* As a user, I want to mark jobs as Applied.
* As a user, I want to track the status of every application.

## Dashboard

* As a user, I want to see all my saved and applied jobs in one place.
