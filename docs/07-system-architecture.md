# System Architecture

## Architecture Style

ApplyForge follows a client-server architecture with a modular backend.

## Technology Stack

### Frontend

* React
* HTML
* CSS
* JavaScript

### Backend

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA

### Database

* PostgreSQL

### Authentication

* JWT (JSON Web Token)

### AI (Future)

* OpenAI / Gemini / Local LLM

## High-Level Flow

1. User logs in.
2. User uploads a resume.
3. Resume is parsed and stored.
4. User imports a job posting.
5. The system compares the resume with the job description.
6. The match result is displayed.
7. The user tracks the application status.
