# Use Cases

## UC-1 User Registration

**Actor:** User

**Flow:**

1. User opens the application.
2. User clicks Register.
3. User enters details.
4. System creates the account.
5. User is redirected to the login page.

---

## UC-2 Upload Resume

**Actor:** User

**Flow:**

1. User logs in.
2. User navigates to Resume.
3. User uploads a PDF or DOCX file.
4. System stores the resume.
5. System extracts resume information.

---

## UC-3 Match Resume with Job

**Actor:** User

**Flow:**

1. User imports a job posting.
2. System reads the job description.
3. System compares it with the user's resume.
4. System displays:

   * Match percentage
   * Matching skills
   * Missing skills

---

## UC-4 Track Job Application

**Actor:** User

**Flow:**

1. User opens a saved job.
2. User marks the application status.
3. System updates the dashboard.
